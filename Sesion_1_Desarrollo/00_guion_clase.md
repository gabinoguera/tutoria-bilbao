# Guía Sesión 1: Introducción a GitHub Copilot y Primeros Pasos

**Fecha:** Martes 04/11/2025  
**Horario:** 9:00 a 10:30h (90 minutos)  
**Modalidad:** Online  
**Cliente:** Metro Bilbao

## Presentación (5 min)
Bienvenida. Explicar que Copilot es un asistente de código, no magia. Hoy aprenderemos lo básico: cómo pedirle cosas y validar que funcionen.

## Objetivos de la Clase
- Entender qué es GitHub Copilot y cómo usarlo
- Aprender a hacer prompts simples y efectivos
- Generar código básico con ayuda de Copilot
- Validar que el código generado es correcto

## Estructura y Guion de la Clase

### 1. Introducción a GitHub Copilot (15 min)
- ¿Qué es Copilot? Demo rápida
- Cómo hacer preguntas a Copilot (Copilot Chat)
- Buenas prácticas básicas: ser claro, dar contexto
- **Archivo:** `01_introduccion_copilot.md`

### 2. Ejercicio 1: Primeros Prompts - Calculadora Simple (20 min)
- Ejercicio SUPER SIMPLE: Crear función de suma
- Paso a paso guiado
- Validar el resultado
- Iterar si no funciona
- **Archivo:** `02_ejercicio_primeros_prompts.md`

### 3. Ejercicio 2: SQL Básico - Leer y Explicar (20 min)
- Copiar una query SQL simple en un archivo
- Pedirle a Copilot "Explain This"
- Pedirle a Copilot que genere una query básica
- Discutir los resultados
- **Archivo:** `03_ejercicio_sql_simple.md`
- **NO NECESITAMOS BASE DE DATOS** - Solo leer y discutir SQL

### 4. Ejercicio 3 (OPCIONAL): Arreglar un Error (10 min)
- **Solo si hay tiempo**
- Código JavaScript con error simple
- Pedirle a Copilot que lo identifique y arregle
- **Archivo:** `04_ejercicio_debugging_simple.md`
- **Si no hay tiempo:** Saltar directamente al cierre

### 5. Cierre y Tarea (10 min)
- Recap de lo aprendido
- Tarea simple: Probar Copilot con una función de su trabajo
- Preview Sesión 2: Documentación y SQL
- Q&A

## Notas para el Instructor

### ⚠️ CRÍTICO: Este Grupo es Básico
- Van LENTOS - dar mucho más tiempo del planificado
- Se pierden fácilmente - repetir conceptos
- Necesitan ver CADA PASO - no asumir nada
- Asistir individualmente si se atascan

### Tips de Facilitación:
1. **Ir MUY despacio** - Mejor terminar 2 ejercicios bien que 4 mal
2. **Mostrar TU pantalla** - Que vean exactamente qué hacer
3. **Pausas frecuentes** - "¿Todos van aquí? ¿Alguien necesita ayuda?"
4. **No usar jerga técnica** - Explicar todo en términos simples
5. **Celebrar pequeños logros** - "¡Genial, funcionó!"

### Prioridades Si Falta Tiempo:
